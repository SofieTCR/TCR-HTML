Opdracht 2.2 Onderzoeken

Waarom noem je de eerste HTML-pagina; index.html?

Omdat alle browsers altijd als eerste voor een bestand genaamd index zoeken.

Met welke HTML versie werken we nu?

Wij werken met HTML 5.

Met welke CSS versie werken we nu?

Wij werken met CSS 4.15.

Hoe kan het dat een website er net iets anders uit ziet in een andere browser? (Edge, Chrome, Firefox, Opera)

Doordat somige browsers net andere afstanden tussen letters of regels gebruiken en oudere browsers geen CSS gebruiken.

Welke browser is het populairst en hoeveel procent RAM gebruikt deze browser?

De meest populaire browser is Google Chrome en deze browser gebruikt 3.2% van mijn RAM met 12 tabbladen open.
